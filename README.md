#Это стартовый пакет для приложения

**В пакете:**

1. Ангуляр (только база)
2. gulp (gulp, gulp-rigger, gulp-watch)
3. sass(сконфигурирована компиляция компосом)  
подключены хелперы, щрифты, вариаблес бутстрапа, произведена сброска стилей
4. подключен fontAwesome



После установки выполнить bower install и npm install
